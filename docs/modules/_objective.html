Module _objective

BZ98R LUA Extended API Objective Manager.

Note that this module cannot manage manually created objectives.

Usage:

    local objective = require("_objective");

Info:

  • Author: John "Nielk1" Klein

Class ObjectiveManager

ObjectiveManager:UpdateObjective (name[, color[, duration[, text[, position[, persistant]]]]])
Updates the objective message with the given name. If no objective exists with that name, it does nothing.

Parameters:

  • name string Unique name for objective, usually a filename ending with otf from which data is loaded
  • color ColorLabel Default to "WHITE". (optional)
  • duration number defaults to 8 seconds (optional)
  • text string Override text from the target objective file. [2.0+] (optional)
  • position number Sort position of the objective. Defaults to the next available ID. (optional)
  • persistant boolean If true, the objective will not be removed when the objectives are cleared. Defaults to false. (optional)

ObjectiveManager:RemoveObjective (name)
Removes the objective message with the given file name. Messages after the removed message will be moved up to fill the vacancy. If no objective exists with that file name, it does nothing.

Parameters:

  • name string

ObjectiveManager:ReplaceObjective (oldname, name[, color[, duration[, text[, position[, persistant]]]]])
Replaces the objective message with the given name. If no objective exists with that name, it does nothing.

Parameters:

  • oldname string Unique name for objective, usually a filename ending with otf from which data is loaded
  • name string Unique name for objective, usually a filename ending with otf from which data is loaded
  • color ColorLabel Default to "WHITE". (optional)
  • duration number defaults to 8 seconds (optional)
  • text string Override text from the target objective file. [2.0+] (optional)
  • position number Sort position of the objective. Defaults to the removed objective's position or next available ID. (optional)
  • persistant boolean If true, the objective will not be removed when the objectives are cleared. Defaults to false. (optional)
