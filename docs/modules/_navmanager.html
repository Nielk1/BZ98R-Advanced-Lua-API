Events

NavManager:NavSwap
Nav GameObjects swapped. The old nav will be deleted after this event is called. This event allows for scripts to replace their nav references or copy over custom data.

Call method: CallAllNoReturn

Parameters:

  • old GameObject GameObject instance
  • new GameObject GameObject instance

Class GameObject

GameObject An object containing all functions and data related to a game object.
GameObject:BuildImportantNav (odf, team, location[, point])
Build an important nav and add it to the collection. Important navs will push non-important navs out of the way in the list.

Parameters:

  • odf string ODF of the nav to build, if nil uses the default nav ODF
  • team integer Team number of the nav to build
  • location Vector, Matrix, Handle or string Position vector, ransform matrix, Object, or path name.
  • point integer If the location is a path this is the path point index, defaults to 0. (optional)

Returns:

    GameObject or nil nav The nav object that was built
GameObject.CompactionStrategy
What to do when empty slots exist and excess navs exist

Fields:

  • DoNothing Do nothing, leave excess navs in overflow
  • ChronologicalToGap Excess navs inserted into gaps in order of creation
  • ImportantFirstToGap Excess navs inserted into gaps in order of importance, then creation
  • [1] DoNothing
  • [2] ChronologicalToGap
  • [3] ImportantFirstChronologicalToGap

GameObject:AllNavGameObjects (team[, include_overflow])
Enumerates all navs for a team. At least 10 indexes will be iterated, even if there are no navs in those slots. Navs not in the nav list, known internally as "Overflow Navs", will be returned with indexes above 10.

Parameters:

  • team integer Team number to enumerate
  • include_overflow boolean If true "Overflow Navs" will be included in the enumeration after the initial 10. (optional)

Returns:

  1. integer index The index of the nav in the enumeration
  2. GameObject nav The nav object at the index

Usage:

  • for i, nav in navmanager.AllNavGameObjects(1, true) do
    print("Nav " .. i .. ": " .. tostring(nav));
end
  • local active_navs = utility.IteratorToArray(navmanager.AllNavGameObjects(1));
