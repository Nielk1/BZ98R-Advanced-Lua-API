Utility - Iterator Operations

IteratorToArray (iterator)
Convert an iterator to an array. This function takes an iterator and converts it to an array. It handles both array-like and non-array-like iterators. Sparse numeric indices will result in `nil` values in the array. Duplicate numeric indices will overwrite previous values. An empty iterator will return an empty array.

Parameters:

  • iterator function The iterator to convert

Returns:

    any[] array An array containing the values from the iterator
