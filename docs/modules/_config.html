Module _config

BZ98R LUA Extended API Configuration.

Constants used to configure the API's system. Note that reading any value other than "locked" will lock the config table.

Usage:

    local config = require("_config");

if not config.locked then
    config.hook_priority.DeleteObject.GameObject = -99999;
end
