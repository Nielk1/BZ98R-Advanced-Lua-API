Module _unsaved

BZ98R LUA Extended API Unsaved.

Crude custom type to make data not save/load exploiting the custom type system.

Usage:

    local unsaved = require("_unsaved");

data.unsavable = unsaved(data.unsavable);

Info:

  • Author: John "Nielk1" Klein
