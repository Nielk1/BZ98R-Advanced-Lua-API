CameraObject (base, right, up, forward, target)
Offsets a cinematic camera from a base game object while looking at a target game object. Returns true if the base or handle game object does not exist. Returns false otherwise.

Parameters:

  • base GameObject or Handle
  • right number Meters to the right of the base object. (0.01 resolution)
  • up number Meters above the base object. (0.01 resolution)
  • forward number Meters in front of the base object. (0.01 resolution)
  • target GameObject or Handle

Returns:

    boolean
