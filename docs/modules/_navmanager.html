BuildImportantNav (odf, team, location[, point])
Build an important nav and add it to the collection. Important navs will push non-important navs out of the way in the list.

Parameters:

  • odf string ODF of the nav to build
  • team number Team number of the nav to build
  • location vector position, matrix transform, or string path name to build the nav or nav GameObject to clone and delete
  • point string Path point number for the nav to build on if using a path (optional)

Returns:

  1. GameObject The nav object that was built
  2. number The index of the nav in the NavCollection for the team
MoveImportantNavsUp (team)
Move important navs up in the list. This will move all important navs to the top of the list, pushing unimportant navs down. This works by recreating navs so be sure to re-grab your navs.

Parameters:

  • team number Team number of the nav to move
GetNav (team, index)
Get the nav for a team and index.

Parameters:

  • team number Team number of the nav to get
  • index number Index of the nav to get

Returns:

    GameObject The nav object at the specified index for the team
