Module _optional

BZCC LUA Extended API Optional loader.

Load a lua module optionally.

Usage:

    local optional = require("_optional");
local missingSuccess, missingMod = optional("_missing");
missingMod = missingSuccess and missingMod or nil;

local missing2Success, missingMod2 = require("_optional")("_missing2");
missingMod2 = missing2Success and missingMod2 or nil;

Info:

__call (table, moduleName)
__call Attempt to load a module, if it fails return false and error, if succesful return true and the module.

Parameters:

  • table table The module table itself.
  • moduleName string Module name to load.

Returns:

  1. boolean success True if the module loaded successfully, false if it failed.
  2. The module return value or nil if failed
